# IDEA常用快捷键

==ctrl+alt+t==    	 		  **选中代码放入循环/if/……中**

==ctrl+d==  		 		     **复制光标所在行到下一行**

==ctrl+ +==     	 		    **展开代码**

==ctrl+ -==     	   	  	 **折叠代码**

==ctrl+shift++== 			   **展开所有代码**

==ctrl+shift+-== 		  	  **折叠所有代码**

==ctrl+f==    			    	 **文本查找**

==ctrl+r==     			    	**文本替换**

==alt+F3==   		   		  **选中文本，逐个往下查找相同文本，并高亮显示**

==ctrl+alt+Enter==  	   	**光标所在行上空出一行，光标定位到新行**

==alt+Insert==      	   	  **添加自动生成的代码**

==ctrl+shift+/==   			   **代码块注释**

==shift+F6== 		   		  **选中变量批量修改**

==ctrl/alt+shift+↑==    		 **移动光标所在行向上移动**

==ctrl/alt+shift+↓==   	  	**移动光标所在行向下移动**

==shift+F6==					   **重命名**

==ctrl+alt+Enter==             **向下插入一行**

==shift+Enter==                **向上插入一行**

==ctrl+n==							**从项目中搜索**

==alt+1==							 **显示/隐藏左边项目栏**
